## 💻 实现细节

### 1. 地址缩略函数

```typescript
// 格式化地址（缩略显示）
const formatAddress = (address: string) => {
  if (!address) return '-'
  return `${address.slice(0, 6)}...${address.slice(-4)}`
}
```

**使用示例：**
```typescript
// 输入
formatAddress('0x1234567890abcdef1234567890abcdef12345678')

// 输出
'0x1234...5678'
```

---

### 2. 地址单元格渲染

```tsx
<TableCell>
  <div className="flex items-center gap-2">
    {/* 缩略地址 */}
    <code className="text-xs font-mono">
      {formatAddress(token.wallet_address)}
    </code>
    
    {/* 复制按钮 */}
    <Button
      variant="ghost"
      size="sm"
      className="h-6 w-6 p-0 hover:bg-muted"
      onClick={() => copyAddress(token.wallet_address, '钱包地址')}
    >
      {copiedAddress === token.wallet_address ? (
        <Check className="h-3 w-3 text-green-500" />
      ) : (
        <Copy className="h-3 w-3" />
      )}
    </Button>
  </div>
</TableCell>
```

**样式说明：**
- `text-xs font-mono`：小字号 + 等宽字体
- `h-6 w-6`：紧凑的按钮尺寸
- `hover:bg-muted`：鼠标悬停效果

---

### 3. USD 价值单元格渲染

```tsx
<TableCell className="text-right">
  <span className="font-bold text-[#F0B90B] text-base">
    ${formatNumber(token.usd_value)}
  </span>
</TableCell>
```

**样式说明：**
- `font-bold`：加粗字体
- `text-[#F0B90B]`：币安黄色
- `text-base`：稍大字号（16px）
- `text-right`：右对齐（数字习惯）

---

### 4. 授权状态单元格渲染

<TableCell className="text-center">

**样式说明：**
- **已授权**：
  - `bg-green-500`：绿色背景
  - `hover:bg-green-600`：悬停变深绿
- **未授权**：
  - `variant="destructive"`：警告样式
  - `bg-red-500`：红色背景
  - `hover:bg-red-600`：悬停变深红

## 💡 设计理念

### 1. 信息层次

**优先级排序：**
1. **高优先级**：USD 价值、授权状态
   - 使用**强对比色**（金黄、红、绿）
   - **加粗**字体
2. **中优先级**：代币符号、名称
   - 使用**正常字重**
   - 标准色彩
3. **低优先级**：地址、时间
   - 使用**小字号**
   - **灰色**调

---

### 2. 色彩心理学

| 颜色 | 情感 | 应用 | 效果 |
|------|------|------|------|
| 🟡 金黄 | 财富、价值 | USD 价值 | 吸引注意 |
| 🟢 绿色 | 安全、完成 | 已授权 | 放心忽略 |
| 🔴 红色 | 警告、紧急 | 未授权 | 引起关注 |
| ⚫ 黑色 | 中性、数据 | 其他信息 | 辅助阅读 |

---

### 3. 功能 vs 美观

| 改进 | 功能性 | 美观性 | 平衡 |
|------|-------|--------|------|
| 地址缩略 | ✅ 节省空间 | ✅ 更整洁 | ✅ 最佳 |
| USD 金黄色 | ✅ 快速识别 | ✅ 品牌一致 | ✅ 最佳 |
| 红色未授权 | ✅ 高辨识度 | ✅ 符合认知 | ✅ 最佳 |

## ⚠️ 注意事项

### 1. 地址识别

**缩略格式：**
- 前 6 位 + 后 4 位 = 10 个字符
- 对于大多数场景已足够识别
- 如需完整地址，点击复制按钮

**建议：**
- 可考虑添加悬停显示完整地址
- 或在点击地址时弹出详情

---

### 2. 色彩对比度

**可访问性检查：**
- ✅ 金黄色 (#F0B90B) 在白色背景上对比度良好
- ✅ 绿色徽章有足够对比度
- ✅ 红色徽章有足够对比度

**WCAG 标准：**
- 正常文本：对比度 ≥ 4.5:1 ✅
- 大文本：对比度 ≥ 3:1 ✅

---

### 3. 响应式适配

**小屏幕优化建议：**
- 考虑隐藏次要列（如创建时间）
